
This is a data visualization project I did, using Python in Jupyter Notebook, Matplotlib, Plotly and Pandas. I used a CSV file and applied various visualization and filters to 
gain some insights into the type of customer this company has. 



Shop/customer data set, that includes columns Annual Income, Profession, Spending Score, Family Size.
